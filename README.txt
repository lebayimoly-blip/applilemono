🚗 VehicleApp – Plateforme de gestion intelligente des véhicules
📌 Version Démo – Données fictives
🧠 Présentation
VehicleApp est une application web conçue pour simplifier et automatiser la gestion des flottes de véhicules. Elle s’adresse aux administrations, entreprises, collectivités et gestionnaires de parc automobile.

Cette version démo permet de tester les fonctionnalités principales avec des données fictives.

🎯 Objectifs
Centraliser les informations des véhicules

Suivre les entretiens, incidents et assurances

Gérer les utilisateurs selon des rôles définis

Importer des données via Excel

Visualiser des statistiques en temps réel
